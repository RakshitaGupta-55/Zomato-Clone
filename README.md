# 🍽️ Zomato Clone

Zomato Clone is a responsive web application that replicates the core features of a food delivery platform. Users can browse restaurants, explore menus, and discover food options through a clean and interactive interface.

---

## 🌟 Features

* 🔍 Browse restaurants and food items
* 📋 View menus and details
* 📱 Fully responsive design
* ⚡ Smooth and interactive UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Bootstrap
* Tailwind CSS

---

## 📂 Project Structure

```
Zomato-Clone/
│── index.html
│── css/
│── js/
│── assets/
```

---

## 🚀 Getting Started

1. Clone the repository:

   ```
   git clone https://github.com/RakshitaGupta-55/zomato-clone.git
   ```

2. Navigate to the project folder:

   ```
   cd zomato-clone
   ```

3. Open `index.html` in your browser
